## Features

- [Next.js](https://nextjs.org) App Router
- [Naver Map API](https://www.ncloud.com/product/applicationService/maps)를 활용한 지도 탐색 기능
- 소셜 로그인 기능(카카오, 네이버, 구글)
- [Tailwind CSS](https://tailwindcss.com/)를 활용한 스타일링
